Settings

Configure TypePilot and its direct connection to the Gemini API.

Google Gemini API Key

TypePilot connects directly to Google. No TypePilot server receives your key or selected text.

AI Model

Choose a compatible text model. The list is validated against your API key.

Enter your API key to load available text models.

Model guide: Flash-Lite prioritizes speed and quota efficiency. Gemini 3.5 Flash prioritizes output quality. Gemini 2.5 Flash-Lite remains the compatibility fallback.

Translation

Choose the target used by the Translate action. The action menu updates automatically.

English remains the default. The selected language is stored locally and included only with translation requests.

Writing Instructions

Base instructions applied to Fix and every action in the dropdown menu.

TypePilot enforces the structured response automatically. Do not add JSON formatting instructions.

Personal Dictionary

Protect names, brands, product codes, and specialist terms from unwanted corrections.

Enter one term per line. The list stays on this device and is sent to Gemini only with a writing request.
