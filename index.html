Privacy by design

Your text does not pass through a TypePilot server.

TypePilot uses a Bring Your Own Key model. Your selected text and relevant writing settings travel directly from Chrome to the Google Gemini API only when you request an action.

  • API key and settings stored locally in Chrome.
  • No TypePilot account, analytics, or remote database.
  • No advertising, profiling, or sale of user data.
